方式一：APP 转发联动（推荐，APP/PC 都能同时看到事件）
# PC 端：启动 Web 服务（接收 APP 推送）
cd SVC_PC_View
pip install flask flask-socketio
python server/app.py --no-bridge --port 5000

# 浏览器打开 http://localhost:5000

# 手机端：在 App -> 过滤页 ->「PC 联动转发」里开启
# - Host 填 PC 的局域网 IP（如 192.168.1.10）
# - Port 填 5000
# - 点击「测试连接」确认可用


方式二：ADB Bridge 实时联动（仅 PC 看，依赖端口 9527）
# 终端1：启动 ADB Bridge
python android_bridge/adb_bridge.py

# 终端2：启动 Web 服务
pip install flask flask-socketio
python server/app.py

# 浏览器打开 http://localhost:5000


方式三：离线分析
# 从设备拉取事件文件
adb pull /data/local/tmp/svc_events.jsonl ./

# 离线加载
python server/app.py --no-bridge --load svc_events.jsonl
